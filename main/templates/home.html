О проекте

Tuya Home Online - это универсальный веб-клиент платформы Интернета вещей Tuya Smart. Веб-клиент позволит управлять вашим умным домом с любого пользовательского устройства, вне зависимости от провайдера мобильного приложения.

Главные преимущества веб-приложения Tuya Home Online

  • возможность управлять умным домом с ПК, с любого клиентского устройства помимо смартфона
  • унифицированный интерфейс для всех устройств
  • поддержка всех мобильных прижений платформы (Tuya Smart, Smart Life, DIGMA smart life, BlitzWolf, Я Смарт, HIPER IOT, ELARI SmartHome, HomeAlone, EKF Connect, SLS, DENVER SMART HOME, ЭРА Smart, KOJIMA HOME, GEOZON HOME, IRBIS Home, Perenio Lite, Minimir Home, Gosund, iFEEL Safe+Smart, Sibling, STARWIND SmartLife, Navigator SmartHome, Securic Smart Life, Brilliant Smart, ... )
  • возможность взаимодействия с разработчиком, для развития проекта под потребности пользвоателей

Как работает?

Пользователь получает токен доступа к мобильному приложению на iot.tuya.com -> Пользователь авторизуется на Tuya Home Online с указанным токеном ->
