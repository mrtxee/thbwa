Tuya Home Online - это универсальный веб-клиент платформы Интернета вещей Tuya Smart. Веб-клиент позволит управлять вашим умным домом с любого пользовательского устройства, вне зависимости от провайдера мобильного приложения. Главные преимущества веб-приложения:

Кроссплатформенность

Возможность управлять умным домом с ПК, с любого клиентского устройства помимо смартфона static/img/1055687-crosspl.png

Унифицированный интерфейс

Унифицированный интерфейс для всех устройств умного дома. Отсутсвуе визуальный шума, в сравнении с мобильным приложением

Любой источник данных

Поддержка всех мобильных прижений платформы. Tuya Smart, SmartLiving, DIGMA, BlitzWolf, EKF Connect, ЭРА Smart, iFEEL, IRBIS, STARWIND, Navigator, HIPER, ELARI, HomeAlone, SLS, DENVER, KOJIMA, GEOZON, ЯСмарт, Perenio, Minimir, Gosund, Sibling, Securic, Brilliant...

Поддержка

открытая поддержка пользователей для совместного развития проекта улучшения пользовательского опыта

Как работает?

Пользователь получает токен доступа к мобильному приложению на iot.tuya.com -> Пользователь авторизуется на Tuya Home Online с указанным токеном ->
