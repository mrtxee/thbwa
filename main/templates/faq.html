Tuya Home Online переносит интерфейс управления Вашими умными устройствам из мобильного приложения в веб-интерфейс.

Привязка устройств и программирование сценариев происходит только через Ваше мобильное приложение. Для того, чтобы у tuyahome.online получил доступ к Вашим устройствам, необходимо в профиле пользователя указать реквизиты доступа к сервисам tuya.iot. Endpoint url - регион, который используется в Вашем мобильном приложении. Для генерации реквизитов: Access id, Access secret, UID понадобится мобильное приложение и инструкция ниже.

Как получить Access id, Access secret, UID?
1. Войти в аккаунт tuya
2. Создайте новый проект
3. Подключить мобильное приложение
4. Скопировать реквизиты доступа

Как получить Endpoint url?
В этом поле необходимо выбрать регион из списка, который используется в Вашем мобильном приложении.
